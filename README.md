> [!WARNING]
> Все представленные скрипты созданы в образовательных целях. Перед запуском настоятельно рекомендую изучить код, чтобы понимать, какие изменения вносятся в систему. Автор не несет ответственности за возможные сбои в работе ОС или потерю данных.

> [!NOTE]
> Скрипты с суффиксом `_Q` поддерживают пакетную обработку. Их можно использовать на панели инструментов [Total Commander](https://github.com/wincmd64/blog/wiki/TotalCmd) кнопкой с параметром `%P%S`. Еще вариант, как ярлык в папке `Shell:SendTo` (меню _Отправить_ в **Проводнике**) - это же можно сделать автоматом запустив с ключом `/s`.  
Скрипты с суффиксом `_CM` поддерживают интеграцию в контекстное меню.

### DirGet\
Большинство скриптов в этой папке объединены общими принципами.
Если запустить скрипт без параметров, будет предложено обновить или скачать (по умолчанию, прямо в рабочей директории) соотв. программу с официального источника. Это своего рода альтернатива **winget**, но без регистрации приложений в системе и со своими фишками. Там, где это возможно, реализуется портативность и возможность авто-ассоциации с типами файлов. Дополнительные ключи см. в описании к каждому скрипту. Например, для **Viber** используя ключ `/h` можно блокировать рекламу патчем файла `hosts`.  

### (root)
`LayoutSwitcher.ahk`. Переключает раскладки клавиатуры по горячим клавишам. По `Ctrl+Alt+Shift+F12` - можно посмотреть все коды установленных раскладок и добавить\изменить нужные под себя.

`hash_CM.bat`. Добавляет в контекстное меню файлов пункт `Hash`, позволяющий быстро вычислить контрольные суммы файла.

`officeDL.bat`. Авто-установка M$ Office. Загружает [Office Deployment Tool](https://officecdn.microsoft.com/pr/wsus/setup.exe) и применяет Ваш [.xml конфиг](https://config.office.com/deploymentsettings). Если своего конфига нет, можно воспользоваться встроенным.

`PassFinder_Q.bat`. Демонстрация метода атаки по словарю. Пытается автоматически подобрать пароль к архивам из текстового файла.

`ps_CM.bat`. Упрощает работу с `.ps1` файлами. Добавляет меню с вариантами запуска и настройками политики выполнения сценариев. 

`TBWinPE_CM.bat`. Добавляет в контекстное меню `.wim` файлов команду для прямой загрузки системы из WIM-образа с помощью утилиты TeraByte TBWinPE.

`uploader_Q.bat`. Выгрузка файлов на сервисы временного хостинга (Litterbox или 0x0.st), автоматически копируя полученную ссылку в буфер обмена.

`win10_11_tweaks.bat`. Быстрая настройка Windows. Различает 10 \ 11 со своими твиками. Перед выполнением создается точка восстановления.

`yt-dlp.bat`. Враппер для [yt-dlp](https://github.com/yt-dlp/yt-dlp). Качает по ссылке из буфера обмена, параметра ком.строки или пакетно.

`filter.ps1`. Быстрый фильтр строк текстовых файлов. Удобно для [текстовых таблиц](https://www.tablesgenerator.com/text_tables).

`pass.ps1`. Секьюрный генератор паролей.

`pathman.ps1`. Добавляет \ удаляет каталог в пользовательскую переменную `%PATH%`. Поддерживает пакетный режим.

`schtasks.ps1`. Создает задание в планировщике для запуска приложения\скрипта при входе в систему без запроса UAC.

`uwp-remover.ps1`. Удаляет некоторые встроенные UWP-приложения Windows 11 в один клик.
