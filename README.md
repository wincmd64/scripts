> [!WARNING]
> Все представленные скрипты созданы в образовательных целях. Перед запуском настоятельно рекомендую изучить код, чтобы понимать, какие изменения вносятся в систему. Автор не несет ответственности за возможные сбои в работе ОС или потерю данных.

> [!NOTE]
> Скрипты с суффиксом:  
> `_Q` поддерживают пакетную обработку. Их можно использовать на панели инструментов [Total Commander](https://github.com/wincmd64/blog/wiki/TotalCmd) кнопкой с параметром `%P%S`. Еще вариант, как ярлык в папке `Shell:SendTo` (меню _Отправить_ в **Проводнике**).  
> `_CM` поддерживают интеграцию в контекстное меню.  
> `_eget` используют [eget](https://github.com/inherelab/eget) для установки и обновления.
